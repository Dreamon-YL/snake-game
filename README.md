# 贪吃蛇游戏

这是一个使用 HTML5 Canvas 和 JavaScript 实现的经典贪吃蛇游戏。

## 游戏特点

- 简单直观的操作方式
- 流畅的游戏体验
- 实时分数显示
- 响应式设计

## 如何玩

1. 使用方向键（↑↓←→）或 WASD 键控制蛇的移动
2. 吃到食物可以增加分数和蛇的长度
3. 避免撞到墙壁或蛇自身

## 在线试玩

你可以通过以下链接在线体验游戏：
[点击这里开始游戏](https://[你的GitHub用户名].github.io/snake-game/)

## 本地运行

1. 克隆此仓库
2. 在浏览器中打开 index.html 文件
3. 开始游戏！
